# (C# 코딩) 그림판
## 개요
- C# 프로그래밍 학습
- 1줄 소개: 사용자 키보드 입력을 받아서 처리하는 프로그램

- 사용한 플랫폼: 
	- C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
		- Label, TextBox, ListBox, Button
	- 사용한 기술과 구현한 기능:
		- ui 구성 : 도형선택,색선택,굵기선택,캔버스 구성
		- 도형선택: 버튼 3개를 이용해서 직선, 사각형,원 선택
		- 색선택: combobox를 이용해서 색 선택
		- 선굵기선택: trackbar를 이용해서 선 굵기 선택
		- 캔버스: picturebox를 이용해서 그림 그릴 수 있는 공간 구성
